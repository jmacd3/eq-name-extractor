Eq-name-extractor

Bookmarklet for extracting names from lds.org directory

View project on GitHub

Installing

Adding a the bookmarklet is quite easy. Just open up your browser of choice and create a new bookmark. Name it somehting that you can remember and paste the following in as the url (make sure that you copy the entire js snipet ):


javascript:(function(){s=document.createElement('script');s.type='text/javascript';s.src='https://jmacd3.github.io/eq-name-extractor/javascripts/extractor.js?v='+parseInt(Math.random()*99999999);document.body.appendChild(s);})();

Usage

  1. Log into lds.org
  2. Go to the Directory in the tools menu
  3. Click on the Ward Organizations link
  4. Select the organization you are interested in (the bookmarklet will work with any of them).
Click on you bookmarklet (It might be helpful to put the bookmark in your bookmark bar or somewhere it is easily accesable). If all goes well you should see a new link appear as shown below.
